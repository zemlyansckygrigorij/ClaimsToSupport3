Руководство пользователей

При работе с данной программой вам необходимо .

  1. Выбрать тип Заявку из списка имеющихся ошибок
  2. Проверить пользователя если пользователь указан не верно вбить вручную
  3. Ввести дополнительную информацию о проишествии
  4. Нажать кнопку "отправить заявку"
  5. После звонка тех.поддержки нажать кнопку "тех.поддержка"
  6. После нажатия кнопки "тех.поддержка", необходимо прекратить работу с комьютером в течении 3 мин для правильной отправки данных в тех. поддержку.
  7. После окончания работы тех.поддержки необходимо закрыть программу teamViewer и закрыть данную программу
  8. при появлении дополнительных окон с сообщениями об ошибках необходимо связаться с отделом технического обеспечения
